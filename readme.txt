== Description ==

Shows a directory of authors and their personal info

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `warp-userprofile` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Create a page with the title and url you want. e.g.: *Our Team* in `/team`
1. Go to Options ->	Profiles and Select the page you just created
1. Edit the users you want to be shown and check the *Public profile* checkbox
